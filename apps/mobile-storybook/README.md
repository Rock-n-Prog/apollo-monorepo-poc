# ACME Mobile Storybook App

Storybook Expo app for ACME.

This app uses:

- [`packages/mobile/ui`](../../packages/mobile/ui): UI library

## Setup

```bash
pnpm install
```

## Available scripts

### Run

```bash
pnpm dev

# For device-specific runs
pnpm dev:android
pnpm dev:ios
```

### Build

```bash
pnpm build
```

### Lint

```bash
pnpm lint

# To fix lint
pnpm lint:fix
```
